iMMERSIFLY

Project

iMMERSIFLY is a prototype website which envisions what a futuristic website for a business in the year 2045 would look like. The business specialises in immersive multi-sensory experiences using futuristic immsersive technologies to reflect how digital interaction and everyday life may evolve in the future.

Tools

HTML (Bootstrap)

CSS

Python (Flask)

About iMMERSIFLY

This website was developed for a university project for an experiential design module. "Design a speculative product, system, service, or experience intended for the hybrid city of 2035–2045". As I am a web developer with an interest in immersive technologies it made sense to create a promotional and informational website for a futuristic business in the immersive technologies space.

The website offered three different tiers, each tier using more advanced technology than the previous. The first tier reflected how immersive technologies might look in the year 2035, the second was for 2040 and lastly, the third tier was for the year 2045. Each tier was a different price and for a different purpose.


What I learned

I got to put into practice my skills in HTML (Bootstrap), CSS and Python (Flask) in a fun and creative way. I really enjoy creating fun and engaging websites so this was a great project for that as I was able to have lots of fun with vibrant colour schemes with glowing elements. I also learned more about immersive technologies through research into how they can be used currently and how they might be used in the future are these technologies advance.

As the business wasn't real and I instead had to imagine what a business in that point in the future would look like, I came across many things when developing it requiring further thought and development. For example, I wanted it be somewhat realistic so as I was developing the website I realised there had to be another device to connect everything together which is where the "iHub" idea came from. I also came up with a full backstory/about section with information on how the business came to life starting from 2026-2045. It was also important to me to actually try and address current trends such as remote work and the use of AI which was incorporated into the website.
