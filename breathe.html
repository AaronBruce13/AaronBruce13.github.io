breathe

Project

breathe is an interactive prototype application created in Figma. It's fully interactive and features animations made within Figma as well as Jitter and Lottie animations. It's aimed at providing users with a sense of calm and mindfulness.

Tools

Figma

Jitter

Lottie Animations

About breathe

This project was for my "designing the visual interface" module. I had limited prior experience with Figma and animations so this was a fantastic opportunity to build my skills in these areas. I also usually focus on websites as opposed to smartphone applications so this project helped me expand in that area as well.

The app had several sections that were easy to navigate between. Each section served a different purpose and each design choice was chosen carefully to give that sense of calmness from the colour scheme to the typography and even following neilsens heuristics for example giving the user the freedom to change from light to dark mode.


What I learned

Not only did I increase my prototyping skills within Figma, I also increased my ability to design for mobile applications, create animations, icons. I experimented with Figma's built in tools, particularly for the interactivity and animations like prototyping and etc as well as using other things such as Jitter and Lottie animations.
